## 技术架构
### 前端技术
- HTML5 + CSS3 + JavaScript (ES6+)
- Bootstrap 5.3.0 (UI框架)
- Leaflet 1.9.4 (地图组件)
- Web Components (自定义组件)

### 后端技术
- PHP 8.1.2 (服务端语言)
- SQLite 3 (数据库)
- RESTful API设计
- Session管理

### 部署环境
- PHP内置服务器
- 端口暴露服务
- HTTPS安全连接

## 数据库信息
### 数据表
1. **users** - 用户账户信息
2. **restaurants** - 餐厅基本信息 (8个示例餐厅)
3. **dishes** - 菜品信息 (24个示例菜品)
4. **reviews** - 用户评价 (5个示例评价)

### 示例数据
- 校园食堂、Pizza Corner、马来风味餐厅等8家餐厅
- 椰浆饭、玛格丽特披萨、咖喱鸡等24道菜品
- 用户评价和评分数据
